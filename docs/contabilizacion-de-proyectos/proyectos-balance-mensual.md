# Balance mensual del proyecto

Este tutorial esta enfocado en la verificación del balance mensual de un proyecto. Dentro del balance mensual podrá encontrar todas las transacciones de gastos e ingresos del proyecto, sin IVA.

Los pasos para ver el balance mensual del proyecto son los siguientes:

1. Ir a “Contabilidad”.
2. Seleccionar “Reportes”.
3. Seleccionar “Balance del Proyecto”.

![imagen1](/img/contabilizacion-de-proyectos/proyectos-balance-mensual-1.jpg)



Le aparecerá el balance del proyecto, para cambiar de proyecto, seleccione el proyecto que desea ver y presione cambiar.

El balance se debe leer desde abajo hacia arriba, las transacciones están ordenadas por orden cronológico.

![imagen2](/img/contabilizacion-de-proyectos/proyectos-balance-mensual-2.jpg)
