# Cambiar mi contraseña
Para cambio de contraseña podemos realizarlo desde nuestro perfil con los siguientes pasos:


- Presionar el botón de "Editar perfil"

![editar password1](/img/configuraciones-de-mi-usuario/cambiar-mi-contrasena-1.png)

- Presionar el botón de "Cambiar contraseña"

![editar password2](/img/configuraciones-de-mi-usuario/cambiar-mi-contrasena-2.png)

-	Colocar antigua contraseña
-	Nueva contraseña
-	Confirmar nueva contraseña

![editar password3](/img/configuraciones-de-mi-usuario/cambiar-mi-contrasena-3.png)
