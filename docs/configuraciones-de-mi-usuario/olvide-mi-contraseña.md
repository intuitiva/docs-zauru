# Olvide mi contraseña

Suele suceder que se nos olvida nuestra contraseña, para ingresar de nuevo es necesario realizar los siguientes pasos después de ingresar nuestro usuario y contraseña:

Ingresar a: Olvide mi contraseña
Ingresar a: Usuario
Reiniciar mi contraseña


![olvide password id zauru com](/img/configuraciones-de-mi-usuario/olvide-mi-contraseña-1.png)


Nos llegará un correo en el cual debemos ingresar nuestra nueva contraseña.

![correo olvide password](/img/configuraciones-de-mi-usuario/olvide-mi-contraseña-2.png)
