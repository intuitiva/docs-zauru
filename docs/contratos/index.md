Este módulo está diseñado para llevar control de generación automática de documentos recurrentes como ordenes de venta, facturas, ordenes de compra y casos de soporte.

Los contratos en Zauru permiten:
- Definir la periodicidad y cantidad de documentos a generar automáticamente.
- Usar fórmulas dinámicas para calcular montos en cada cuota (ideal para préstamos, leasing, servicios recurrentes).
- Configurar moras por pagos atrasados.
- Consolidar múltiples cuotas en un solo documento.
- Asociar documentos adicionales como transacciones, envíos, facturas y casos.

## Secciones del Módulo

El módulo de contratos se divide en las siguientes secciones:

1. **Configuración** — Configurar parámetros generales como el item de mora, término de pago para moras y referencia extra en documentos generados.
2. **Contratos Preliminares** — Crear, editar y activar contratos. Los contratos preliminares no generan documentos automáticamente.
3. **Contratos Activos** — Contratos que están generando documentos recurrentemente. Permite generar documentos manualmente, ver estado, moras y detalle de partidas contables por cuota.
4. **Contratos Cerrados** — Contratos que ya finalizaron su ciclo de generación de cuotas. Se pueden re-abrir si es necesario.
5. **Reportes** — Reportes de estado consolidado, contratos con generación pendiente, contratos por expirar y contratos pendientes de cierre.
