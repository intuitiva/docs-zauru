Este tutorial esta enfocado en las configuraciones del punto de venta para poder emitir facturas y crear nuevos casos, las configuraciones que se pueden hacer son para predeterminar ciertos valores que usted desea que siempre se generen en un nuevo caso o en una nueva factura, como por ejemplo, si usted desea que cada factura que se emita genere impuestos de una forma predeterminada, o no.

Los pasos para configurar el punto de venta son los siguientes:

1. Ir a “P.D.V”.
2. Seleccionar el icono de “Configuraciones”.


Le aparecerán las opciones de configuraciones para el punto de venta, los campos que se pueden colocar son para predeterminar ciertos valores en el punto de venta, y son los siguientes:

a. Coloque si cada vez que se emite una nueva factura, esta estará sujeta a impuestos o no predeterminadamente.

b. Coloque un agencia predeterminada para el modulo de punto de venta, o déjelo en blanco si el empleado ya tiene una agencia asignada.

c. Coloque si desea tener un termino de pago predeterminado cada vez que se emita una factura.

d. Coloque si desea tener un cliente predeterminado cada vez que se emita una nueva factura.

En la parte inferior le aparecerán las opciones para configurar los valores predeterminados para nuevos casos, los que se pueden configurar son:

e.  Coloque si cada vez que se emite un nuevo caso, este estará sujeta a impuestos o no predeterminadamente.

f. Coloque si desea tener una agencia predeterminada en los casos del punto de venta, o déjela en blanco si cada empleado ya tiene su agencia asignada.

g.  Coloque este cheque si desea que aparezcan los términos de pago seleccionables al crear un nuevo caso, o déjelo en blanco para que no aparezcan.

h. Coloque si desea tener un termino de pago predeterminado para todos los nuevos casos que se creen.

i. Coloque si desea tener una cantidad de días predeterminados para los nuevos casos.


Por ultimo, para guardar sus configuraciones presione “Actualizar Configuraciones”.
