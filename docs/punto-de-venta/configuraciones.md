Este tutorial esta enfocado en las configuraciones del punto de venta para poder emitir facturas, crear ordenes de venta, casos, compras, reservas, traslados y recibir mercaderia. Las configuraciones que se pueden hacer son para predeterminar ciertos valores que usted desea que siempre se generen en los nuevos documentos del punto de venta.

Los pasos para configurar el punto de venta son los siguientes:

1. Ir a "P.D.V".
2. Seleccionar el icono de "Configuraciones".

![imagen1](/img/punto-de-venta/configuraciones-1.jpg)


Le apareceran las opciones de configuraciones para el punto de venta, organizadas en secciones. Los campos que se pueden configurar son los siguientes:

## Seccion: Facturas y Ordenes

a. **Cliente predeterminado para facturas y ordenes**: Seleccione un cliente que se asignara automaticamente cada vez que se cree una nueva factura u orden de venta.

b. **Agencia predeterminada para facturas y ordenes**: Seleccione la agencia (punto de venta) predeterminada donde se emitiran las facturas y ordenes. Si el empleado tiene una agencia asignada, esta tendra prioridad.

c. **Sujeto a impuestos por defecto (facturas y ordenes)**: Active esta opcion si desea que cada nueva factura u orden de venta este sujeta a impuestos de forma predeterminada. Puede desactivar la visibilidad de esta opcion para que el vendedor no la modifique.

d. **Ocultar opcion de sujeto a impuestos**: Si activa esta opcion, el campo de "sujeto a impuestos" no sera visible para el vendedor al crear facturas. Se utilizara el valor predeterminado configurado.

e. **Termino de pago predeterminado para facturas y ordenes**: Seleccione el termino de pago que se asignara por defecto a las nuevas facturas y ordenes de venta.

f. **Permitir seleccionar termino de pago en facturas**: Active esta opcion si desea que el vendedor pueda seleccionar el termino de pago al crear una factura. Si se desactiva, se usara el termino de pago predeterminado.

g. **Transito por defecto en facturas**: Active esta opcion si desea que las facturas nuevas tengan el transito activado por defecto, permitiendo gestionar entregas y envios.

h. **Termino de pago por agencia**: Para cada agencia (punto de venta), puede configurar un termino de pago especifico que se utilizara en esa agencia.

i. **Metodo de pago predeterminado por agencia**: Para cada agencia, configure el metodo de pago que aparecera seleccionado por defecto al momento de cobrar.

## Seccion: Configuraciones de visualizacion

j. **Mostrar marcas**: Active esta opcion para que los productos se puedan filtrar por marca en el punto de venta.

k. **Mostrar imagenes de productos**: Active esta opcion para que se muestren las imagenes de los productos en el catalogo del punto de venta.

l. **Botones de ancho doble**: Active esta opcion para mostrar los botones de productos con el doble de ancho en la interfaz del punto de venta.

m. **Ordenar productos por disponibilidad en lugar de nombre**: Active esta opcion para ordenar los productos por su disponibilidad en lugar de por nombre alfabetico.

n. **Mostrar codigo en lugar de nombre**: Active esta opcion para que se muestre el codigo del producto en lugar del nombre al crear facturas.

o. **Mostrar referencia en facturas nuevas**: Active esta opcion para que el campo de referencia este visible al crear una nueva factura.

p. **No mostrar impresion de factura**: Active esta opcion si NO desea que se muestre la pantalla de impresion automaticamente despues de crear una factura.

q. **Mostrar boton "agregar por codigo"**: Active esta opcion para que aparezca un boton que permita agregar productos por codigo en facturas, ordenes, casos y compras.

r. **Mostrar precios de la ultima emision para el cliente**: Active esta opcion para mostrar los precios de la ultima factura emitida al mismo cliente como referencia.

s. **Precios finales editables**: Active esta opcion para permitir editar los precios finales de cada linea en lugar de los precios unitarios cuando se usan precios flexibles.

t. **Codigo de barras en iOS**: Active esta opcion si utiliza dispositivos iOS para escanear codigos de barras.

u. **Excluir filtro de busqueda**: Active esta opcion para excluir el filtro de busqueda en algun contexto del punto de venta.

v. **Mostrar transporte en facturas**: Active esta opcion para que se muestre la opcion de "necesita transporte" en las facturas.

w. **Ocultar acciones de detalles, anular e imprimir de otros usuarios**: Active esta opcion para que los vendedores no puedan ver detalles, anular o imprimir facturas y ordenes creadas por otros usuarios.

## Seccion: Casos (servicio tecnico)

x. **Sujeto a impuestos por defecto (casos)**: Active esta opcion si desea que cada nuevo caso este sujeto a impuestos de forma predeterminada.

y. **Agencia predeterminada para casos**: Seleccione la agencia predeterminada donde se registraran los casos.

z. **Cliente predeterminado para casos**: Seleccione el cliente que se asignara automaticamente a los nuevos casos.

aa. **Termino de pago predeterminado para casos**: Seleccione el termino de pago que se asignara por defecto a los nuevos casos.

ab. **Permitir seleccionar termino de pago en casos**: Active esta opcion si desea que el vendedor pueda seleccionar el termino de pago al crear un caso.

ac. **Dias de cierre predeterminados**: Especifique la cantidad de dias despues de la creacion en que se espera cerrar el caso.

ad. **Mostrar boton "agregar por codigo" en casos**: Active esta opcion para mostrar el boton de agregar por codigo al crear casos.

## Seccion: Compras

ae. **Agencia predeterminada para compras**: Seleccione la agencia donde se registraran las compras.

af. **Sujeto a impuestos por defecto (compras)**: Active esta opcion si desea que cada nueva orden de compra este sujeta a impuestos de forma predeterminada.

ag. **Ocultar opcion de sujeto a impuestos en compras**: Active esta opcion para ocultar el campo de sujeto a impuestos en las ordenes de compra.

ah. **Proveedor predeterminado para compras**: Seleccione el proveedor que se asignara por defecto a las nuevas ordenes de compra.

ai. **Termino de cargo predeterminado para compras**: Seleccione el termino de cargo que se asignara por defecto a las nuevas ordenes de compra.

aj. **Permitir seleccionar termino de cargo en compras**: Active esta opcion si desea que se pueda seleccionar el termino de cargo al crear una compra. Por defecto viene activado.

ak. **Costos finales editables en lugar de costos unitarios**: Active esta opcion para permitir editar los costos finales en lugar de los costos unitarios en las lineas de compra.

al. **Mostrar boton "agregar por codigo" en compras**: Active esta opcion para mostrar el boton de agregar productos por codigo al crear compras.

am. **Mostrar descuento en compras**: Active esta opcion para mostrar el campo de descuento en las ordenes de compra.

an. **Mostrar fecha de envio en compras**: Active esta opcion para mostrar el campo de fecha de envio en las ordenes de compra.

ao. **Mostrar referencia en lineas de compras de gastos**: Active esta opcion para mostrar un campo de referencia en cada linea de las compras de gastos.

ap. **Termino de cargo por agencia**: Para cada agencia de compras, configure un termino de cargo especifico.

aq. **Metodo de descargo predeterminado por agencia**: Para cada agencia, configure el metodo de descargo (forma de pago a proveedores) predeterminado.

ar. **No establecer transito en destino**: Para cada agencia de compras, configure si desea que NO se genere un transito automatico al recibir productos hacia esa agencia.

## Seccion: Traslados

as. **Origen de reabastecimiento predeterminado por agencia**: Para cada agencia de punto de venta, configure la agencia de origen predeterminada desde donde se solicitan los traslados de productos.

at. **Ocultar otras agencias en solicitudes de traslado**: Para cada agencia, puede activar esta opcion para que solo se muestre la agencia de origen configurada como predeterminada en las solicitudes de traslado, ocultando las demas.

## Seccion: Otras configuraciones

au. **Filtrar agencias por categoria en reservas**: Active esta opcion para que al crear una reserva solo se muestren las agencias de la misma categoria que la agencia de origen.

av. **Mostrar boton "agregar por codigo" en reservas**: Active esta opcion para mostrar el boton de agregar productos por codigo al crear reservas.

aw. **Busqueda de clientes en servicio externo**: Active esta opcion para permitir la busqueda de clientes utilizando un servicio externo de almacenamiento de documentos (factura electronica) en facturas, ordenes y casos.

ax. **Mostrar telefono del cliente**: Configure si desea mostrar y permitir editar el telefono del cliente, o si desea que el telefono funcione como campo de busqueda de clientes en facturas, ordenes y casos.

ay. **Impresion Star WebPRNT**: Si cuenta con impresoras termicas Star Micronics con servidor web, active esta opcion y configure el endpoint de impresion para cada usuario.

az. **Carga de imagenes**: Active las opciones para permitir la carga de imagenes en ordenes de venta, facturas, casos, compras, compras de gastos, reservas, y la carga de archivos PDF en compras.

Por ultimo, para guardar sus configuraciones presione "Actualizar Configuraciones".

![imagen2](/img/punto-de-venta/configuraciones-2.png)

Le aparecera un mensaje de exito en la pantalla notificandole que se actualizaron los datos.
