## Modulos del Punto de Venta

El punto de venta (P.D.V.) de Zauru ofrece las siguientes funcionalidades:

### Configuracion
- **[Configuraciones](configuraciones.md)**: Configure los valores predeterminados para facturas, ordenes, casos, compras, traslados y preferencias de visualizacion del punto de venta.

### Operaciones de venta
- **[Crear orden de venta](crear-orden-de-venta.md)**: Cree ordenes de venta con reserva de productos, para luego convertirlas en facturas.
- **[Crear factura](crear-factura.md)**: Emita facturas directamente desde el punto de venta, con soporte para lector de codigo de barras.
- **[Cobrar una factura o una orden de venta](cobrar-una-factura-o-una-orden-de-venta.md)**: Registre los cobros de facturas y ordenes de venta, con soporte para multiples metodos de pago.

### Servicio tecnico
- **[Crear caso](crear-caso.md)**: Registre casos de servicio tecnico con productos, servicios y diagnosticos.
- **[Cobrar caso](cobrar-caso.md)**: Cobre los casos de servicio tecnico creados desde el punto de venta.

### Operaciones de compra e inventario
- **[Crear compra](crear-compra.md)**: Registre ordenes de compra de productos o gastos.
- **[Recibir compra](recibir-compra.md)**: Reciba los productos de las ordenes de compra para que ingresen al inventario.
- **[Crear reserva](crear-reserva.md)**: Reserve productos en una bodega para su traslado a otra sucursal.
- **[Solicitar traslado](solicitar-traslado.md)**: Solicite productos a una bodega central u otra sucursal para reabastecer su punto de venta.
- **[Gestionar envios](gestionar-envios.md)**: Administre los envios en transito y registre las entregas de productos.

### Reportes
- **[Reportes](reportes.md)**: Consulte cierres diarios, ventas mensuales, productos vendidos y otros reportes operativos del punto de venta.
