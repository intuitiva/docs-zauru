# Permisos de Acceso
Zauru maneja un esquema de permisos muy poderosos. Cumple con 2 propósito principales:
1. Limitar o permitir el acceso a usuarios a áreas con información sensible para sus responsabilidades.
2. Establece flujos de trabajo para los usuarios, de tal manera que solo va a poder ejecutar permisos que tenga.
