Este módulo sirve para el control de dinero, esto incluye:

1. Cuentas por cobrar.
2. Cuentas por pagar.
3. Bancos.
4. Conciliaciones bancarias.
5. Presupuestos.
6. Libros fiscales (libro de ventas, libro de compras, libro diario, libro mayor, estado de resultados, balance general).
7. Centro de costos.
