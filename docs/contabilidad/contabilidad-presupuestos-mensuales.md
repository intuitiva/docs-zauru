# Presupuestos mensuales

Este tutorial esta enfocado en la creación de un presupuesto mensual y un presupuesto mensual por proyecto. Cuando usted hace un presupuesto, detalla la cantidad que debería gastar por cada rubro contable, si usted pasa su presupuesto, le aparecerá en rojo el rubro en los detalles del presupuesto.

### Presupuesto mensual normal
Los pasos para crear un presupuesto mensual son los siguientes:

1. Ir a “Contabilidad”.
2. Seleccionar “Presupuestos”.
3. Seleccionar “Nuevo Presupuesto”.

![imagen1](/img/contabilidad/contabilidad-presupuestos-mensuales-1.jpg)


Le aparecerán las opciones para crear un nuevo presupuesto, los campos a llenar son los siguientes:

a. Coloque el mes del presupuesto.

b. Coloque el año del presupuesto.

c. Especifique si es un presupuesto de gastos o ingresos.

d. Coloque la cantidad total a presupuestar.

e. Aquí es donde deberá detallar el presupuesto. Seleccione la cuenta de gasto o ingreso, y especifique cuanto esta presupuestado para este rubro. Para agregar un nuevo rubro presione “+”, para agregar 2 nuevos rubros presione “+2”.

![imagen2](/img/contabilidad/contabilidad-presupuestos-mensuales-2.jpg)


Especifique la cantidad presupuestada por rubro de gasto o ingreso, y al finalizar presione “Crear presupuesto” para guardar los cambios.

![imagen3](/img/contabilidad/contabilidad-presupuestos-mensuales-3.jpg)


Le aparecerá un mensaje de éxito en la pantalla notificándole que se creo el presupuesto, para ver los detalles de su presupuesto seleccione el icono de “ver” (El ojo).

![imagen4](/img/contabilidad/contabilidad-presupuestos-mensuales-4.jpg)
